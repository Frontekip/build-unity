# build-unity
Command line to build Unity games


## Installation 

To install a stable release use:

    yarn add @frontekip/build-unity -g
    
or use npm:

    npm install @frontekip/build-unity -g


## Quickstart 

Windows Build

    $ build-unity --windows .
    
MacOS

    $ build-unity --macos .
    
    
After these commands, you can look in the `.build` folder.
