# Build Unity Games

Command line to build Unity games


## Installation 

To install a stable release use:

    yarn global add @frontekip/unity
    
or use npm:

    npm install @frontekip/unity -g


## Quickstart 

    unity --help


#### Windows Build

    $ unity build --windows .
    
#### MacOS

    $ unity build --macos .
    
    
After these commands, you can look in the `.build` folder.
